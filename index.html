Browser

Browser detection utils (IE)

Dependencies

  • jQuery - http://jquery.com/

Methods

not_msie():Boolean
Returns true if the current browser is not msie (any version)
is_msie((String) conditional):Boolean
Returns true if the conditional is met. e.g Browser.is(msie(">= 8")
supports_animation
Returns true if the current browser supports animation (Is equals or greater than IE10)
